Truck Search
Created a web application that allows for searching the Department of Transportation's data from a sales perspective:
  • Wrote web scraper HTML data from 10,000,000 possible records
  • Extracted, Transformed and Loaded data into my postgres database
  • Used Django back-end framework
  • Braintree for creditcard processing
  • Deployed to AWS using: RDS, Elastic Beanstalk, Codecommit, CodeBuild, CodePipeline
